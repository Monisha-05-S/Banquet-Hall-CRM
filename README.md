

# Banquet Hall Booking CRM

##  Project Overview

This project is a Customer Relationship Management (CRM) application developed using Salesforce to manage banquet hall bookings efficiently. It helps in organizing customer details, managing bookings, tracking billing, and preventing duplicate reservations.

---

##  Features

* Customer Management
* Banquet Hall Management
* Booking System with validation
* Billing and Payment Tracking
* Automation using Flow
* Reports and Dashboard visualization

---

##  Technologies Used

* Salesforce CRM
* Flow Builder
* Apex Trigger 

---

##  Modules

* Customer
* Banquet Hall
* Booking
* Billing
